# StreamVolume Guard 0.1.4

Version publique de test centrée sur la stabilisation finale du rendu audio, la lisibilité des guides testeurs et la cohérence des distributions multi-navigateurs.

## Points clés

- Pipeline audio stabilisé pour rapprocher les sons faible, fort et très fort après activation de l'extension.
- Critère streamer verrouillé : environ `-21 dB RMS` traité et `-18 dB Peak OBS estimé` après stabilisation.
- Page de test locale améliorée avec un design cohérent avec la page Options.
- Checklist mainteneur ajoutée pour préserver le contrat audio validé.
- README clarifié pour guider les débutants vers les fichiers `.zip` des releases GitHub.
- Distributions Chromium, Firefox, Firefox Android et source Safari régénérées avec les guides utiles aux testeurs.

## Fichiers recommandés

- Projet complet : `streamvolume-guard-project-0.1.4.zip`
- Chrome, Brave, Edge desktop : `streamvolume-guard-chromium-0.1.4.zip`
- Firefox desktop : `streamvolume-guard-firefox-0.1.4.zip`
- Firefox Android : `streamvolume-guard-firefox-android-0.1.4.zip`
- Safari macOS / iOS / iPadOS : `streamvolume-guard-safari-source-0.1.4.zip`

## Confidentialité

StreamVolume Guard reste open source, sans tracker, sans télémétrie automatique et sans collecte d'audio, d'historique ou de données personnelles.
